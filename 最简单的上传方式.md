# 🚀 最简单的上传方式（不用命令行）

## 方式一：GitHub 网页直接上传（推荐）

### 步骤 1：创建仓库
1. 打开 https://github.com/new
2. 输入仓库名：`pc-commander`
3. 选择 **Public**
4. 点击 **Create repository**

### 步骤 2：网页上传文件
1. 在新创建的仓库页面，点击 **"Add file"**
2. 选择 **"Upload files"**
3. 拖拽整个 `PC` 文件夹到上传区域
4. 等待上传完成（可能需要几分钟）
5. 点击 **"Commit changes"**

### 步骤 3：等待构建
1. 点击上方的 **"Actions"** 标签
2. 等待构建完成（3-5分钟）
3. 点击 **"Releases"** 标签
4. 下载 APK！

---

## 方式二：使用一键脚本

### 步骤 1：安装 Git
1. 下载：https://git-scm.com/download/win
2. 安装（一直点 Next 就行）

### 步骤 2：运行脚本
1. 双击运行 `一键上传到GitHub.bat`
2. 输入你的 GitHub 用户名
3. 输入仓库名（默认：pc-commander）
4. 自动上传！

---

## 方式三：GitHub Desktop（图形界面）

### 步骤 1：下载 GitHub Desktop
https://desktop.github.com/

### 步骤 2：添加本地仓库
1. 打开 GitHub Desktop
2. File → Add local repository
3. 选择 `PC` 文件夹
4. 点击 **Publish repository**
5. 登录 GitHub 账号
6. 完成！

---

## 📱 上传后获取 APK

无论哪种方式，上传后：

1. 访问 `https://github.com/你的用户名/pc-commander/actions`
2. 等待构建完成（绿色勾）
3. 访问 `https://github.com/你的用户名/pc-commander/releases`
4. 下载 APK！

---

## 💡 推荐

- **完全不懂技术** → 用方式一（网页拖拽上传）
- **愿意安装软件** → 用方式二（一键脚本）
- **喜欢用图形界面** → 用方式三（GitHub Desktop）

---

**选择一种方式，5分钟后就能下载 APK！** 🚀
