### 方式三：真机 App（完整功能）

#### 步骤 1：安装 HBuilderX

1. 下载地址：https://www.dcloud.io/hbuilderx.html
2. 下载对应系统的版本（Windows 选 `HBuilderX.3.x.x.zip`）
3. 解压到任意目录

#### 步骤 2：导入项目

1. 打开 HBuilderX
2. 点击 `文件` → `导入` → `从本地目录导入`
3. 选择：`c:\Users\33455\Desktop\PC\PC_Commander_Product\Mobile_App_Source`

#### 步骤 3：配置服务器地址

打开 `Mobile_App_Source/pages/config/api.vue`，修改默认地址：

```javascript
// 找到第 85 行左右
config: {
    serverUrl: 'http://你的电脑IP:8000',  // ← 修改这里
    openCodePath: 'opencode',
    language: 'zh-CN'
}
```

**获取电脑 IP**：
```bash
# 在 PowerShell 中运行
ipconfig | findstr "IPv4"
# 输出类似：192.168.1.5
```

#### 步骤 4：真机调试

1. **开启手机 USB 调试**：
   - 设置 → 关于手机 → 连续点击"版本号"7次开启开发者模式
   - 设置 → 开发者选项 → 开启 "USB 调试"

2. **连接手机**：用数据线连接手机和电脑

3. **运行到手机**：
   - HBuilderX 顶部菜单 → `运行` → `运行到手机或模拟器` → 选择你的设备
   - 首次会自动安装 HBuilder 基座应用

4. **等待运行**：手机上会出现 App，可以实时调试

#### 步骤 5：打包 APK（可选）

如果想发给朋友使用：

1. HBuilderX → `发行` → `原生App-云打包`
2. 勾选 `Android` → 使用 `DCloud 公共证书`
3. 点击 `打包`
4. 等待 2-5 分钟，下载 APK 安装包

## 🐛 常见问题排查

### 问题 1：手机无法连接 PC 服务器

**症状**：App 显示"连接失败"

**排查步骤**：
1. 检查 PC 和手机是否在同一 WiFi
2. 检查 PC 防火墙是否放行 8000 端口
3. 尝试关闭 Windows 防火墙测试
4. 检查 IP 地址是否正确

```bash
# 检查端口是否监听
netstat -an | findstr 8000

# 临时关闭防火墙（测试用）
netsh advfirewall set allprofiles state off
```

### 问题 2：HBuilderX 检测不到手机

**症状**：运行菜单没有显示手机设备

**解决方案**：
1. 确认 USB 调试已开启
2. 更换 USB 线（有些线只能充电）
3. 安装手机驱动（Windows 设备管理器查看）
4. 尝试重启 HBuilderX 和手机

### 问题 3：App 安装后闪退

**排查**：
1. 检查 `manifest.json` 配置
2. 查看 HBuilderX 控制台错误日志
3. 确保 `pages.json` 路径正确

### 问题 4：跨域错误（CORS）

**症状**：浏览器控制台显示 CORS 错误

**解决**：服务器已配置 CORS，如仍有问题：
```javascript
// server.js 中添加
app.use((req, res, next) => {
    res.header('Access-Control-Allow-Origin', '*');
    res.header('Access-Control-Allow-Methods', 'GET, POST, PUT, DELETE, OPTIONS');
    res.header('Access-Control-Allow-Headers', 'Content-Type, Authorization');
    next();
});
```

## 📊 测试环境要求

### PC 端
- **Node.js**: >= 16.0.0
- **操作系统**: Windows 10/11, macOS, Linux

### 手机端
- **Android**: >= 8.0 (API 26)
- **iOS**: >= 12.0（需 Mac + Xcode 打包）
