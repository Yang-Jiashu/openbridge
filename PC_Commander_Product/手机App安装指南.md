# 📱 PC Commander - 手机 App 安装指南

## 🎯 安装方式（2种）

### 方式一：真机调试（推荐，适合开发测试）
**特点**：实时修改、即时生效、无需反复打包  
**时间**：5-10分钟

### 方式二：打包 APK（适合发给朋友使用）
**特点**：生成独立安装包、可离线安装  
**时间**：10-20分钟

---

## 方式一：真机调试（推荐）

### 步骤 1：下载 HBuilderX

1. 访问官网：https://www.dcloud.io/hbuilderx.html
2. 下载 **Windows 标准版**（约 300MB）
3. 解压到任意位置（无需安装）

```
下载地址：https://download.dcloud.net.cn/HBuilderX.4.45.2025010502.zip
```

### 步骤 2：开启手机 USB 调试

#### Android 手机设置：
1. **打开开发者模式**：
   - 设置 → 关于手机 → 连续点击"版本号"7次
   - 会提示"您已处于开发者模式"

2. **开启 USB 调试**：
   - 设置 → 系统 → 开发者选项
   - 开启 "USB 调试"
   - 开启 "USB安装"（部分手机需要）

3. **连接电脑**：
   - 使用 USB 数据线连接手机和电脑
   - 手机上选择 "传输文件" 或 "MTP" 模式

### 步骤 3：导入项目

1. **打开 HBuilderX**
2. **导入项目**：
   - 文件 → 导入 → 从本地目录导入
   - 选择文件夹：`c:\Users\33455\Desktop\PC\PC_Commander_Product\Mobile_App_Source`

3. **等待加载**：
   - 项目会显示在左侧项目管理器中

### 步骤 4：运行到手机

1. **选择运行设备**：
   - 顶部菜单：运行 → 运行到手机或模拟器
   - 选择你的设备（如：HUAWEI P40 Pro）

2. **首次运行**：
   - 会自动下载安装 HBuilder 基座 App
   - 手机上可能提示"是否允许安装"，点击允许
   - 等待基座安装完成

3. **查看效果**：
   - 手机上会自动打开 App
   - 你现在看到的是实时运行的效果

### 步骤 5：配置服务器地址

1. **在 HBuilderX 中打开**：
   - `pages/config/api.vue`

2. **修改默认地址**（第 85 行）：
   ```javascript
   config: {
       serverUrl: 'http://你的电脑IP:8000',  // 修改这里
       // ...
   }
   ```

3. **获取电脑 IP**：
   ```bash
   # 在 PowerShell 中运行
   ipconfig | findstr "IPv4"
   # 输出：192.168.1.5
   ```

4. **修改后保存**，手机会自动刷新

### 步骤 6：测试连接

1. **在手机上**：
   - 进入"API 配置"页面
   - 确认服务器地址正确
   - 点击"保存并测试"

2. **预期结果**：
   - 显示"连接成功"提示

## 方式二：打包 APK 安装包

### 步骤 1：准备打包环境

1. **确保项目能正常运行**（方式一成功后再打包）
2. **修改应用信息**（可选）：
   - 打开 `manifest.json`
   - 修改应用名称、版本号等

### 步骤 2：云打包

1. **HBuilderX 顶部菜单**：
   - 发行 → 原生App-云打包

2. **配置打包选项**：
   ```
   Android 包名：com.opencode.remote
   证书：使用 DCloud 公用证书（免费）
   渠道包：不需要
   ```

3. **点击"打包"**：
   - 首次需要登录 DCloud 账号（免费注册）
   - 等待云端打包完成（约 2-5 分钟）

4. **下载 APK**：
   - 打包完成后会显示下载链接
   - 点击下载 APK 文件

### 步骤 3：安装 APK

#### 方法 A：直接安装（简单）
1. 将 APK 文件传到手机
2. 在手机上点击安装
3. 可能需要在设置中允许"安装未知来源应用"

#### 方法 B：ADB 安装（快速）
```bash
# 确保手机已连接并开启 USB 调试
adb devices

# 安装 APK
adb install -r "PC_Commander.apk"
```

## 🔧 常见问题

### 问题 1：HBuilderX 检测不到手机

**症状**：运行菜单没有显示设备

**解决**：
1. 确认 USB 调试已开启
2. 更换 USB 线（有些线只能充电）
3. 安装手机驱动：
   - Windows 设备管理器 → 找到 Android 设备
   - 右键 → 更新驱动程序
4. 重启 HBuilderX 和手机

### 问题 2：安装基座失败

**症状**：提示"安装失败"或"解析错误"

**解决**：
1. 在手机上允许"安装未知来源应用"
2. 设置 → 安全 → 未知来源 → 允许
3. 或者使用 ADB 手动安装基座

### 问题 3：无法连接服务器

**症状**：App 显示"连接失败"

**排查**：
1. **检查 IP 地址**：
   ```bash
   ipconfig
   # 确保输入的是正确的局域网 IP
   ```

2. **检查防火墙**：
   ```bash
   # 临时关闭防火墙测试
   netsh advfirewall set allprofiles state off
   ```

3. **检查端口**：
   ```bash
   # 确保 8000 端口没有被占用
   netstat -ano | findstr :8000
   ```

4. **检查手机和电脑是否同 WiFi**

### 问题 4：App 闪退

**症状**：打开后立刻关闭

**解决**：
1. 查看 HBuilderX 控制台错误信息
2. 检查 `manifest.json` 配置
3. 确保 `pages.json` 路径正确
4. 尝试清除缓存重新运行
