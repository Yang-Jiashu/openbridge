# PC Commander - 安卓 App 打包指南

## 📱 应用介绍

**PC Commander** 是一个强大的远程 PC 控制工具，让你通过手机即可远程控制电脑，执行命令、管理文件、监控系统状态。

### 功能特点
- ✨ **自然语言控制** - 直接说话即可控制电脑
- 📂 **文件管理** - 查看、管理电脑文件
- 💻 **系统监控** - 查看进程、磁盘、系统状态
- ⚡ **快捷指令** - 一键执行常用操作
- 🌐 **远程访问** - 支持局域网和外网访问
- 🎨 **优雅界面** - 暗黑主题，极客风格

## 🛠️ 打包步骤

### 方法一：使用 HBuilderX 云打包（推荐）

#### 1. 下载安装 HBuilderX

1. 访问官网：https://www.dcloud.io/hbuilderx.html
2. 下载 **Windows 版**（标准版即可）
3. 解压到任意位置，双击 `HBuilderX.exe` 启动

#### 2. 导入项目

1. 打开 HBuilderX
2. 点击菜单栏 **文件** → **打开目录**
3. 选择本文件夹 `Mobile_App_Source`
4. 项目会出现在左侧项目列表中

#### 3. 配置 App 信息

1. 打开 `manifest.json` 文件（在根目录）
2. 点击 **基础配置** 选项卡
3. 填写以下信息：
   - **应用名称**：PC Commander
   - **版本名称**：1.0.0
   - **版本号**：100
4. 点击 **图标配置**，上传应用图标（如果没有，可以使用 `static/logo.svg`）
5. 点击 **权限配置**，确保已勾选网络相关权限

#### 4. 云打包

1. 点击顶部菜单栏 **发行** → **原生 App-云打包**
2. 在弹出的窗口中：
   - **选择平台**：Android
   - **包名**：保持默认或自定义（如 `com.yourname.pccommander`）
   - **证书**：选择 **使用 DCloud 公用证书**（免费快速）
   - **广告**：取消勾选（除非你需要广告）
3. 点击 **打包**
4. 等待几分钟（通常在 2-5 分钟）
5. 打包完成后，控制台会显示下载链接
6. 点击链接下载 `.apk` 文件

#### 5. 安装到手机

1. 将下载的 APK 文件传输到安卓手机
2. 在手机上点击安装
3. 如果出现"未知来源"提示，去设置中允许安装即可

### 方法二：使用 Android Studio 本地打包（高级）

如果你需要更多自定义功能，可以使用 Android Studio 进行本地打包。

#### 1. 导出 Android 工程

1. 在 HBuilderX 中，点击 **发行** → **原生 App-本地打包** → **生成本地打包 App 资源**
2. 等待生成完成，会输出到 `unpackage/dist/build/app-plus` 目录

#### 2. 下载 App 离线 SDK

1. 访问：https://nativesupport.dcloud.net.cn/AppDocs/usesdk/android
2. 下载最新版 Android 离线 SDK

#### 3. 导入 Android Studio

1. 解压 SDK，打开 `HBuilder-Integrate-AS` 项目
2. 将生成的 App 资源复制到 `app/src/main/assets/apps/` 目录
3. 配置 `AndroidManifest.xml` 和 `build.gradle`
4. 使用 Android Studio 构建 APK

## 📋 使用说明

### 首次使用

1. **启动 PC 服务端**
   - 在电脑上运行 PC_Server 的 `start.bat`
   - 确保服务启动成功（显示 `Server running at http://0.0.0.0:8000`）

2. **配置服务器地址**
   - 打开手机 App
   - 点击右上角 **⚙️ 设置**
   - 输入电脑的 IP 地址（如 `http://192.168.1.5:8000`）
   - 点击 **测试连接**，确保显示"已连接"
   - 点击 **保存设置**

3. **开始控制**
   - 返回首页，输入指令或点击快捷操作
   - 例如："查看当前目录文件"、"查看系统信息"

### 获取电脑 IP 地址

1. 在电脑上按 `Win + R`，输入 `cmd`，回车
2. 输入命令：`ipconfig`
3. 找到 **IPv4 地址**（如 `192.168.1.5`）
4. 完整地址格式：`http://192.168.1.5:8000`

### 远程访问（外网）

如果你想在外面（使用 4G/5G）控制家里电脑：

1. 在电脑上安装 [Ngrok](https://ngrok.com/)
2. 运行命令：`ngrok http 8000`
3. 复制生成的 HTTPS 地址（如 `https://xxxx.ngrok-free.app`）
4. 在手机 App 中填入这个地址

## 🔧 常见问题

### Q: 连接失败怎么办？

A: 请检查以下几点：
1. ✅ 电脑和手机是否连接同一个 WiFi
2. ✅ PC 服务端是否已启动
3. ✅ IP 地址是否正确（注意要加 `http://` 前缀）
4. ✅ 防火墙是否拦截了 8000 端口

### Q: 如何清除聊天记录？

A: 进入 **设置** → **清除聊天记录** 即可。

### Q: 支持 iOS 吗？

A: 目前主要优化了 Android。iOS 需要使用 Mac 电脑配合 Xcode 打包，且需要苹果开发者账号。

### Q: 可以自定义快捷指令吗？

A: 可以！修改 `pages/index/index.vue` 中的 `quickActions` 数组即可添加自己的快捷指令。
