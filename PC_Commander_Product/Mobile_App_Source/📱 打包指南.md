## 🛠️ 打包步骤

### 第一步：下载 HBuilderX

1. 访问官网：https://www.dcloud.io/hbuilderx.html
2. 下载 **Windows 标准版**（约 300MB）
3. 解压到任意位置（免安装）
4. 双击 `HBuilderX.exe` 启动

### 第二步：导入项目

1. 打开 HBuilderX
2. 点击 **文件** → **打开目录**
3. 选择 `Mobile_App_Source` 文件夹
4. 项目会出现在左侧项目列表

### 第三步：配置 App 信息

1. 点击项目根目录的 `manifest.json`
2. 配置以下信息：
   - **应用名称**：PC Commander
   - **版本名称**：1.0.0
   - **版本号**：100
3. 点击 **图标配置**
   - 上传 `static/logo.png.svg` 或自定义图标
   - 建议使用 1024x1024 的 PNG 图片
4. 点击 **权限配置**，确保已勾选：
   - ✅ Internet
   - ✅ Network State
   - ✅ WiFi State

### 第四步：云打包（推荐）

1. 点击顶部菜单栏 **发行** → **原生 App-云打包**
2. 配置打包选项：
   - **平台**：Android ✅
   - **包名**：`com.yourname.pccommander`
   - **证书**：使用 DCloud 公用证书（免费）
   - **渠道包**：不需要勾选
3. 点击 **打包** 按钮
4. 等待 2-5 分钟（控制台会显示进度）
5. 打包完成后，点击下载链接获取 APK

### 第五步：安装到手机

1. 将 APK 文件传输到安卓手机
2. 在文件管理器中找到 APK
3. 点击安装
4. 如遇"未知来源"提示：
   - 去 **设置** → **安全** → 允许安装未知来源应用

## 📋 使用说明

### 首次使用

#### 1️⃣ 启动 PC 服务端
```bash
# 在电脑上
cd PC_Server
start.bat
```
确认看到：`Server running at http://0.0.0.0:8000`

#### 2️⃣ 获取电脑 IP 地址
1. 按 `Win + R`，输入 `cmd`，回车
2. 输入：`ipconfig`
3. 找到 **IPv4 地址**（如 `192.168.1.5`）

#### 3️⃣ 配置手机 App
1. 打开 App，等待启动页完成
2. 进入设置页面（或自动跳转）
3. 输入服务器地址：`http://192.168.1.5:8000`
4. 点击 **测试连接**
5. 显示"已连接"后，点击 **保存设置**

#### 4️⃣ 开始远程控制
- 返回首页
- 点击快捷操作按钮，或输入自然语言指令
- 例如："查看当前目录"、"运行计算器"

## 🌐 远程访问（外网）

如果你想在外面（使用 4G/5G）控制家里电脑：

### 使用 Ngrok

1. 下载 [Ngrok](https://ngrok.com/) 并注册账号
2. 获取 Authtoken 并配置
3. 运行命令：
   ```bash
   ngrok http 8000
   ```
4. 复制生成的 HTTPS 地址（如 `https://abc123.ngrok-free.app`）
5. 在手机 App 中填入这个地址
6. ✅ 现在可以在任何地方控制你的电脑了！

## 🔧 常见问题

### Q1: 连接失败怎么办？

检查清单：
- [ ] 电脑和手机在同一 WiFi 网络
- [ ] PC 服务端已启动（看到绿色文字）
- [ ] 输入的 IP 地址正确（包含 http:// 和端口）
- [ ] 防火墙没有拦截 8000 端口
- [ ] 手机有网络访问权限

### Q2: 如何更新服务器地址？

点击首页右上角的 **⚙️ 设置按钮**，或设置页面的返回键重新配置。

### Q3: 聊天记录保存在哪里？

聊天记录自动保存在手机本地存储中（最近 50 条），清除缓存会删除记录。

### Q4: 支持 iOS 吗？

目前主要适配 Android。iOS 版本需要：
- Mac 电脑
- Xcode 开发环境
- 苹果开发者账号（$99/年）

### Q5: 打包失败怎么办？

1. 检查 `manifest.json` 是否配置完整
2. 确认应用 ID 是唯一的（建议修改 appid）
3. 检查图标尺寸是否符合要求
4. 尝试重新登录 HBuilderX 账号

## 🎨 自定义主题

如果你想修改配色方案，编辑以下文件中的 CSS：

### 修改主色调
编辑 `pages/index/index.vue` 中的颜色变量：
```css
/* 主色调 - 紫色渐变 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 背景色 */
background: linear-gradient(135deg, #1a1f3a 0%, #0d1225 50%, #1a0f2e 100%);
```

### 修改快捷指令
编辑 `pages/index/index.vue` 中的 `quickActions` 数组：
```javascript
quickActions: [
    { icon: '📂', name: '查看文件', command: 'ls', gradient: 'linear-gradient(135deg, #667eea 0%, #764ba2 100%)' },
    // 添加你的自定义指令
]
```
