# 🎨 API 配置页面 UI 升级说明

## ✨ 全新赛博朋克风格设计

### 🌟 主要改进

#### 1. 动态背景效果
- **网格地面**: 3D 透视网格，带有流动动画
- **漂浮粒子**: 20+ 个动态粒子随机漂浮
- **发光光斑**: 3 个渐变色光斑，呼吸动画
- **扫描线**: 复古 CRT 显示器扫描效果

```css
.grid-floor {
    transform: perspective(500px) rotateX(60deg);
    animation: gridMove 20s linear infinite;
}
```

#### 2. 霓虹发光主题
- **主色调**: 青色 (#00d4ff) + 紫色 (#7c3aed)
- **强调色**: 橙色 (#f97316) + 绿色 (#10b981)
- **发光效果**: box-shadow + text-shadow 多层叠加
- **渐变动画**: 按钮和卡片的流光效果

#### 3. 状态仪表盘
- **环形指示器**: 带旋转外圈的发光圆环
- **连接线**: 节点间的流光连接线
- **状态文字**: ONLINE/OFFLINE/ERROR 彩色标识
- **点击跳转**: 点击节点滚动到对应配置

#### 4. 卡片设计升级
- **四角边框**: 赛博朋克风格的切角设计
- **展开动画**: 平滑的高度过渡
- **颜色区分**: 
  - OpenCode = 蓝紫色
  - OpenClaw = 橙红色  
  - OpenAI = 绿色
- **发光边框**: 启用时的霓虹光晕

#### 5. 开关组件
- **轨道发光**: 开启时的渐变流光
- **拇指动画**: 弹性滑动效果
- **核心光点**: 内部发光指示器
- **颜色适配**: 根据主题色变化

#### 6. 输入框设计
- **底部光条**: 聚焦时的流光动画
- **外发光**: 激活状态的霓虹阴影
- **渐变背景**: 深色半透明 + 渐变叠加

#### 7. 模型选择器
- **芯片设计**: 圆角标签样式
- **激活状态**: 发光边框 + 背景色
- **悬停效果**: 微妙的颜色变化

#### 8. 按钮效果
- **渐变背景**: 多色渐变流光
- **扫描动画**: 斜向光带移动
- **悬停发光**: 阴影扩散效果
- **成功/错误状态**: 颜色变化

#### 9. 弹窗设计
- **框架边框**: 四角装饰边框
- **状态图标**: 发光圆环 + 对勾/叉号
- **背景模糊**: backdrop-filter 毛玻璃
- **颜色区分**: 成功绿色 / 错误红色

#### 10. Toast 提示
- **顶部滑入**: 平滑的入场动画
- **进度条**: 底部倒计时指示
- **渐变背景**: 蓝绿渐变
- **自动消失**: 2.5秒后淡出

## 🎨 视觉层次

### Z-Index 层级
```
0: 动态背景
1: 内容区域
5: 卡片边框角
10: 导航栏
100: Toast/弹窗
```

### 阴影层级
```
卡片: 0 0 40px rgba(color, 0.05)
按钮: 0 0 30px rgba(color, 0.3)
开关: 0 0 10px color
文字: 0 0 20px rgba(color, 0.5)
```
