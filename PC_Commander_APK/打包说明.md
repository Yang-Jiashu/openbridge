## 🚀 打包方式（3种）

### 方式一：一键打包脚本（推荐）⭐

双击运行 **`一键打包APK.bat`**，按提示操作：

```
[1] 调试版 APK (Debug) - 快速测试
[2] 发布版 APK (Release) - 正式使用
[3] 安装到手机 (需要USB调试)
```

### 方式二：Android Studio

1. **安装 Android Studio**
   - 下载：https://developer.android.com/studio

2. **导入项目**
   - File → Open → 选择 `PC_Commander_APK` 文件夹
   - 等待 Gradle 同步完成

3. **构建 APK**
   - Build → Build Bundle(s) / APK(s) → Build APK(s)
   - 或点击工具栏的 "Build" 按钮

4. **获取 APK**
   - APK 生成在：`app/build/outputs/apk/debug/app-debug.apk`

### 方式三：命令行

```bash
cd PC_Commander_APK

# 构建 Debug APK
./gradlew assembleDebug

# 构建 Release APK
./gradlew assembleRelease
```

## 🔧 环境配置

### 1. 安装 Java JDK

```bash
# 下载并安装 JDK 17
# https://adoptium.net/temurin/releases/?version=17

# 验证安装
java -version
```

### 2. 配置环境变量

**Windows:**
```
JAVA_HOME = C:\Program Files\Eclipse Adoptium\jdk-17
ANDROID_HOME = C:\Users\用户名\AppData\Local\Android\Sdk

PATH 添加：
%JAVA_HOME%\bin
%ANDROID_HOME%\platform-tools
```

### 3. 安装 Android Studio

1. 下载并安装 Android Studio
2. 打开 SDK Manager
3. 安装 Android SDK Platform 33
4. 安装 Android SDK Build-Tools

## 📱 安装到手机

### 方法 1：直接传输
1. 把生成的 APK 文件传到手机
2. 在手机上点击安装
3. 允许"未知来源"安装

### 方法 2：ADB 安装
```bash
adb install PC_Commander_v1.0.0.apk
```

### 方法 3：USB 调试安装
1. 手机开启 USB 调试
2. 连接电脑
3. 运行 `一键打包APK.bat` → 选择 [3]

## 🔑 签名说明

- **Debug 版本**：自动使用调试签名
- **Release 版本**：脚本会自动创建签名密钥
  - 密钥文件：`keystore.jks`
  - 密码：`123456`
  - 别名：`pccommander`

如需使用自己的签名，替换 `keystore.jks` 并修改 `app/signing.properties`。

## 🐛 常见问题

### Q: "JAVA_HOME 未设置"
**解决：** 安装 JDK 并配置 JAVA_HOME 环境变量

### Q: "ANDROID_HOME 未设置"
**解决：** 安装 Android Studio，在设置中查看 SDK 路径，配置 ANDROID_HOME

### Q: "Gradle 同步失败"
**解决：**
1. 检查网络连接
2. 使用国内镜像，修改 `settings.gradle`：
```gradle
pluginManagement {
    repositories {
        maven { url 'https://maven.aliyun.com/repository/public' }
        google()
        mavenCentral()
    }
}
```

### Q: "构建失败"
**解决：**
1. 检查 Android SDK 是否完整
2. 在 Android Studio 中打开，查看详细错误
3. 更新 Gradle 插件版本

### Q: 安装时提示"解析错误"
**解决：**
1. 允许"未知来源"安装
2. 设置 → 安全 → 未知来源 → 允许
