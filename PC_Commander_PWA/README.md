# 📱 PC Commander PWA

这是一个**纯网页版**的手机 App，**无需安装**，直接在浏览器中使用！

## ✨ 特点

- **无需下载**：手机浏览器直接打开就能用
- **添加到主屏幕**：可以像原生 App 一样添加到手机桌面
- **离线可用**：配置保存在本地，无需网络
- **响应式设计**：完美适配手机屏幕
- **功能完整**：AI聊天、终端、配置一应俱全

## 🚀 使用方式

### 方式一：直接打开（最简单）

1. **手机浏览器**打开 `index.html` 文件
2. 即可使用！

### 方式二：添加到主屏幕（推荐）

#### iPhone Safari:
1. 用 Safari 打开 `index.html`
2. 点击底部 **分享按钮**（中间那个）
3. 选择 **"添加到主屏幕"**
4. 完成！现在像打开 App 一样点击图标

#### Android Chrome:
1. 用 Chrome 打开 `index.html`
2. 点击菜单（三个点）
3. 选择 **"添加到主屏幕"**
4. 完成！

## 📱 功能说明

### 首页
- 显示服务器连接状态
- 快速操作入口
- 最近活动记录

### AI 助手
- 发送消息给 PC
- 快捷命令按钮
- 实时显示回复

### 终端
- 直接执行命令
- 显示命令输出
- 支持常用命令

### 配置
- 设置服务器地址
- **中英文切换**
- 测试连接
- API 日志

## 🔧 首次使用配置

1. 点击"配置"页面
2. 输入 PC 服务器地址：
   ```
   http://192.168.x.x:8000
   ```
3. 点击"测试连接"
4. 保存配置

## 💡 提示

1. **服务器地址**：使用你电脑的局域网 IP，如 `http://192.168.1.5:8000`
2. **保持服务器运行**：PC 上的服务器必须一直运行
3. **同一网络**：手机和电脑必须在同一 WiFi 下
4. **添加到主屏幕**：可以获得最好的体验，像原生 App 一样
