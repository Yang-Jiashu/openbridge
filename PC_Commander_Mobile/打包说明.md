# 📦 PC Commander Mobile - 打包说明

## 🎯 打包方式

### 方式一：开发调试（推荐）
**时间**: 10-15分钟  
**适合**: 开发测试

### 方式二：生成 APK
**时间**: 20-30分钟  
**适合**: 发布使用

---

## 准备工作

### 1. 安装 Node.js

https://nodejs.org/ 下载 LTS 版本

```bash
node --version  # v16+
npm --version   # v8+
```

### 2. 安装 JDK

https://adoptium.net/ 下载 OpenJDK 11 或更高版本

```bash
java --version  # 11+
```

### 3. 安装 Android Studio

https://developer.android.com/studio

安装后配置:
- SDK Platforms: Android 13.0 (API 33)
- SDK Tools: Android SDK Build-Tools, Android SDK Command-line Tools

### 4. 配置环境变量

```bash
# Windows 环境变量
ANDROID_HOME = C:\Users\用户名\AppData\Local\Android\Sdk

# PATH 添加
%ANDROID_HOME%\platform-tools
%ANDROID_HOME%\cmdline-tools\latest\bin
```

## 方式一：开发调试

### 步骤 1: 安装依赖

```bash
cd PC_Commander_Mobile
npm install
```

### 步骤 2: 连接手机

1. 手机开启 USB 调试
   - 设置 → 关于手机 → 连续点击版本号7次
   - 设置 → 开发者选项 → 开启 USB 调试

2. 连接手机到电脑

3. 验证连接
```bash
adb devices
# 应该显示设备列表
```

### 步骤 3: 启动应用

```bash
# 启动 Metro 服务器
npm start

# 新终端，安装到手机
npm run android
```

应用会自动安装到手机并启动！

## 方式二：生成 APK

### 步骤 1: 配置签名（可选但推荐）

生成签名密钥:
```bash
cd android/app
keytool -genkeypair -v -storetype PKCS12 -keystore my-upload-key.keystore -alias my-key-alias -keyalg RSA -keysize 2048 -validity 10000
```

配置 `android/gradle.properties`:
```properties
MYAPP_UPLOAD_STORE_FILE=my-upload-key.keystore
MYAPP_UPLOAD_KEY_ALIAS=my-key-alias
MYAPP_UPLOAD_STORE_PASSWORD=你的密码
MYAPP_UPLOAD_KEY_PASSWORD=你的密码
```

### 步骤 2: 构建 Release

```bash
cd android
./gradlew assembleRelease
```

Windows:
```bash
cd android
gradlew.bat assembleRelease
```

### 步骤 3: 获取 APK

APK 位置:
```
android/app/build/outputs/apk/release/app-release.apk
```

### 步骤 4: 安装 APK

```bash
adb install android/app/build/outputs/apk/release/app-release.apk
```

或者将 APK 传到手机安装。

## 🚀 自动化脚本

创建 `build-apk.bat`:
```batch
@echo off
echo ========================================
echo    Building PC Commander APK
echo ========================================
echo.

:: Install dependencies
echo [1/4] Installing dependencies...
call npm install
if errorlevel 1 (
    echo Failed to install dependencies
    pause
    exit /b 1
)

:: Clean build
echo [2/4] Cleaning previous build...
cd android
call gradlew clean

:: Build release
echo [3/4] Building release APK...
call gradlew assembleRelease
if errorlevel 1 (
    echo Build failed
    pause
    exit /b 1
)
cd ..

:: Copy APK
echo [4/4] Copying APK...
set APK_SOURCE=android\app\build\outputs\apk\release\app-release.apk
set APK_DEST=PC_Commander_v1.0.0.apk

copy /Y "%APK_SOURCE%" "%APK_DEST%"

echo.
echo ========================================
echo    Build Complete!
echo ========================================
echo.
echo APK location: %CD%\%APK_DEST%
echo.
pause
```

## 📋 常见问题

### 问题 1: `gradlew` 不是内部或外部命令

**解决**: 添加 Gradle 到 PATH
```bash
# 在 android 目录下使用完整路径
.\gradlew assembleRelease
```

### 问题 2: SDK 找不到

**解决**: 检查 ANDROID_HOME 环境变量
```bash
echo %ANDROID_HOME%
# 应该显示 SDK 路径
```

### 问题 3: 内存不足

**解决**: 修改 `android/gradle.properties`
```properties
org.gradle.jvmargs=-Xmx4096m
```

### 问题 4: 依赖下载慢

**解决**: 配置国内镜像
修改 `android/build.gradle`:
```gradle
allprojects {
    repositories {
        maven { url 'https://maven.aliyun.com/repository/public' }
        google()
        mavenCentral()
    }
}
```
